-Instruções de compilação
 -Em sistemas Unix:
   Através do terminal, aplicar uma compilação no diretório e executar o arquivo Merge.
   
   #gcc "Merge Sort.c" -o Merge
   #./Merge
   
   Após isso, cria-se o arquivo "Nome.Do.Arquivo.Ordenado.txt", a partir do arquivo arquivo desordenado.
   
   OBS.: onde estiver "Arquivo_a_ser_ordenado" alterar pelo nome do arquivo que se quer ordenar, garantindo-se que está no mesmo diretório.
